# SocketTaskManager

A multi-client, socket-based task management server built in **pure Java** using **clean layered architecture** and **SQLite (JDBC)** for persistence.

This project was built to deeply understand:
- Backend system design
- Stateful vs stateless components
- Concurrency and shared state
- Raw JDBC (no ORM, no frameworks)
- Manual dependency injection
